## Audit Notes

- `frontend/src/App.tsx` confirms the real shell has moved beyond runtime mock-band fallback, but it still synthesizes parts of the product model that should eventually come from backend truth.
- `frontend/src/redesign/pages/PlayerPage.tsx` is the clearest remaining honesty gap: it looks productized, but the transport, tab asset, and stem behavior are still local-only.
- `backend/app/main.py` already contains more real product capability than the current UI exposes, which is why the program should be executed as vertical slices instead of another rewrite.
