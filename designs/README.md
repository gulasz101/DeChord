# DeChord Design Prototypes

This directory contains five standalone mocked frontend prototypes for reviewing possible DeChord product directions before refactoring the production frontend.

## Run a prototype

From any design directory:

```bash
bun install
bun run dev
```

Each app is configured to run on port `3001`, so start them one at a time from their own directory.

## Notes

- Every design is intentionally self-contained.
- The five prototypes share product intent, not implementation dependencies.
- All data is mocked.
- The goal is design review, not backend integration.
